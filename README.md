# Excel Project: Data Wrangling for Alternative Fuel Vehicles Market
**Training:** Correlation One - Data Analytics Program   

## Project Description
This project analyzes the Alternative Fuel Vehicles (AFV) market in the United States, with a specific focus on characterizing market trends, energy efficiency patterns, and geographical distribution. The analysis provides insights into the evolving AFV landscape to support data-driven decision making.

## Business Context
Business Problem: Characterize the AFV market in the US and Washington state by analyzing vehicle efficiency trends over time, geographical distribution patterns, and market evolution.

Analytical Challenge: Wrangle and transform multi-source vehicle data to create actionable insights for stakeholders interested in the alternative fuel vehicle sector.

## Data Sources
The analysis utilizes four primary datasets:

**sales:** Vehicle sales and registration data

**models:** Vehicle model specifications and attributes

**range_data:** Vehicle range and efficiency metrics

**ev_data:** Electric vehicle-specific information

## Tools & Techniques
**1) Primary Tool:** Microsoft Excel

**2) Key Functions:** VLOOKUP, MID, TRIM, LEN, VALUE, RIGHT

**3) Analysis Features:** PivotTables, PivotCharts, Histograms

**4) Data Techniques:** Data cleaning, standardization, merging, and transformation

## Key Steps
**1) Data Preparation & Cleaning:**
Cleaned raw data across all four tabs
Standardized column formats and naming conventions
Handled missing values and inconsistent entries

**2) Data Transformation:**
Extracted model and range specifications using text functions (MID, RIGHT)
Converted text to numerical values using VALUE function
Applied TRIM and LEN for data quality assurance

**3)  Data Integration:**
Merged multiple datasets using VLOOKUP operations
Created unified views for comprehensive analysis
Established relationships between vehicle models and their attributes

**4) Analysis & Visualization:**
Built PivotTables to summarize key metrics
Created PivotCharts for trend analysis
Developed histograms for distribution analysis
Generated state-level and temporal visualizations
