# CorrelationOne_Excel_DataWrangling
# Excel Project: Data Wrangling for Alternative Fuel Vehicles Market
**Training:** Correlation One - Data Analytics Program  
**Tool:** Microsoft Excel  

## Project Description
             -Understanding the alternative fuel vehicles market in the US.
-Business Problem:  provide a characterization of the AFV market in the US and in Washington, including data about the energy efficiency of the vehicles over time, geographical distribution, etc.
-Analytical Context:  working with the vehicles.xlsm Excel file that contains four tabs of raw data. The first one is the sales tab, which contains aggregated sales figures for the U.S. The second one is the models tab, which contains data on the EV type. The third one is the range_data tab, which has data about ranges of electric vehicles. These data files come from the US Department of Energy. The last one is the ev_data tab, which is a census of all the electric vehicles in circulation in Washington state. This census data comes from the Washington State Department of Licensing.
## Tools & Techniques
-Tools: Excel Filters, Excel functions (VLOOKUP, MID, TRIM, LEN, VALUE, RIGHT), PivotTables, Charts. 
## Insights
- Energy efficiency trends of AFVs over time
- Market distribution across U.S. states
- Detailed view of electric vehicles registered in Washington State
